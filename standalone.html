🔎 SearchandResearch — Standalone

single HTML file · no server · no dependencies

Paste a URL, scrape it in your browser, and download the results as a ZIP.

Scrape a URL

Pure-browser scraping fetches the page's static HTML only (no JavaScript-rendered content) and is subject to CORS. For JS-heavy sites or saving into the repo's collections/ folder, use the Python dashboard (python app.py).

How to use it (click a step to expand)

1 · Open this file
Double-click standalone.html — it runs entirely in your browser. Nothing to install, no server to start.
2 · Scrape a page
Paste a URL and press Scrape. The page is fetched and parsed; you'll see its title, link count, and text length. Add custom fields (CSS selectors) to pull specific bits of the page.
3 · If a site won't load (CORS)
Browsers block many cross-origin fetches. If you see a fetch/CORS error, put a proxy prefix in the CORS proxy box (for example https://corsproxy.io/?url=) and scrape again. Note: a proxy sees the URLs you fetch — only use one you trust.
4 · Download the results
Download ZIP gives you a folder per page containing page.html, data.json, text.txt, links.json, plus a report.html at the root. Download report.html grabs just the readable report. The ZIP is built in your browser — nothing is uploaded anywhere.
5 · When to use the full Python dashboard instead
Use python app.py for JavaScript-rendered sites, full-page screenshots, crawling links to a depth, respecting robots.txt, or saving straight into the repo's collections/ folder.
